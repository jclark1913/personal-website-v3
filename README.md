### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jclark1913/personal-website-v3.git
   ```
3. Install NPM packages
   ```sh
   npm i
   ```
4. Run the development server
   ```sh
   npm run dev
   ```
